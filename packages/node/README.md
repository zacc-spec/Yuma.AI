# Node.js Dependencies Reference
# This folder tracks npm/Node.js package management

## Management

All Node.js dependencies are managed in `../../frontend/package.json`

To install frontend dependencies:
```bash
cd frontend
npm install
```

To add new packages:
```bash
cd frontend
npm install package-name
```

## Key Dependencies

See `frontend/package.json` for complete list of:
- React/Vite setup
- Build tools
- Development dependencies
